Build your first RAG application

To run these commands, first install the required applications via Manage Apps on a device

Create a new directory and navigate to it

mkdir rag_quickstart && cd rag_quickstart

Download the RAG application script

curl -L -o rag.py https://raw.githubusercontent.com/HPInc/ZGX-Toolkit/main/samples/rag.py

Pull the required Ollama models

ollama pull llama3.2 && ollama pull nomic-embed-text

Download a sample PDF document

curl -L -o document.pdf https://kaas.hpcloud.hp.com/pdf-public/pdf_10584042_en-US-1.pdf

Run the Streamlit application

conda run -n zgx --no-capture-output streamlit run rag.py
